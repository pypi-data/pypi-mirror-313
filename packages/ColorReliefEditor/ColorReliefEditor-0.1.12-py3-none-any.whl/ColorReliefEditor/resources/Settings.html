This tab manages general application settings.

Settings

  • Mode: Mode can be set to basic or expert.
  • You will need to restart for changes to mode or instructions to take effect.
  • Expert mode offers full control over all settings
  • Expert mode supports the use of multiple layers, with each layer having its own elevation files but all layers sharing the same render settings.

Expert Settings

  • Download: URL for a site to download Digital Elevation Files
  • US High Res Download: URL for a site to download US High Resolution files
  • Viewer: Choose the app to use to view images.
